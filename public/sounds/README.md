# Sound Effects

Add the following sound files to this directory:

| File | Usage | Suggested Duration |
|------|-------|-------------------|
| `success.mp3` | RSVP confirmed, action completed | 0.3-0.5s |
| `error.mp3` | Error occurred | 0.3-0.5s |
| `notification.mp3` | New message, notification | 0.2-0.4s |
| `level-up.mp3` | Level up celebration | 1-2s |
| `achievement.mp3` | Achievement unlocked | 1-1.5s |
| `click.mp3` | Button click feedback | 0.1-0.2s |
| `message.mp3` | New message received | 0.2-0.3s |

## Recommended Sources (Free)

- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)

## Volume Guidelines

- Keep sounds at -10dB to -6dB
- Use consistent volume across all sounds
- Test on both speakers and headphones
